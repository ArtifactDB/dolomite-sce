Save and write SingleCellExperiments in Python¶

Introduction¶

The dolomite-sce package is the Python counterpart to the alabaster.sce R package, providing methods for saving/reading SingleCellExperiment objects within the dolomite framework. Currently it’s just a stub but we hope to full functionality soon.
