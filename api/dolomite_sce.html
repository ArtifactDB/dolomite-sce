dolomite_sce.read_single_cell_experiment.read_single_cell_experiment(path: str, metadata: dict, **kwargs) → SingleCellExperiment[source]¶

Load a SingleCellExperiment from its on-disk representation.

This method should generally not be called directly but instead be invoked by read_object().

  • path – Path to the directory containing the object.

  • metadata – Metadata for the object.

  • kwargs – Further arguments, ignored.

Returns:

A SingleCellExperiment with file-backed arrays in the assays.

Method for saving SingleCellExperiment objects to their corresponding file representations, see save_object() for details.

  • x – Object to be staged.

  • path – Path to a directory in which to save x.

  • assay_args – Further arguments to pass to the save_object method for the assays.

  • rdim_args – Further arguments to pass to the save_object method for the reduced dimensions.

  • alt_expts_args – Further arguments to pass to the save_object method for the alternative experiments.

  • kwargs – Further arguments, ignored.
